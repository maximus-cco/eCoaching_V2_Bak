ECR CALLFLOW LOGIC
---------------------
APPLICATION DEFAULTS:
- This application is a "Networx ECR billing" application
- This application MUST ONLY HAVE TNT and ANNOUNCE CONNECT BILLABLE ECR FEATURES ENABLED!!
- This application MUST NOT exit with RLT, EXCEPT if/when extending
  call as a "Type3" (8xx), and the DAP then returns Jump AppID/EP info.
  These calls will be re-initiated as a "Type1" (DDD) extension,
  allowing for RLT!!!
- Callplan Header Requirements:
  Entry Code MUST BE set to "46"
  Billing Type MUST BE set to "3(ICDR,Networx)"
- Ringtimer = 36 seconds (approx. 6 rings)
- Timeout = 5 seconds
- Caller hears call progress tones
- All Call extensions to be set Type 1 (DDD) with RLT
- If BNAR or jump failure: Play Message 6; End call

===============================================================================================================

Caller dials 855-879-2683       (Medicaid)


INITIALIZE CCR AND VARIABLES    CCR_String = DNIS(sys.MSP.DNIS.number) 
                                + Charge Party ANI(sys.MSP.CN.number)
                                + Calling Party ANI(sys.MSP.CgPN.number) 
                                + Entry Point(sys.application_suffix)
                                + OLI (sys.MSP.orig_line_info) 
                                 NOTE: fields are separated by commas
                                Set CallStartDateTime = Current Date / Time 
                                 (Eastern Time YYMMDDHHMMSS)
                                Set SKILLCODE = "FFM_ET1".                               ***ADD 06/14/16***
                                Set SKILLCODE_CCR = "FFM_ET1".                           ***ADD 06/14/16***
                                Go to CHECK ENTRY POINT.


CHECK ENTRY POINT               Check Entry Point (EP).
 If EP=000 (Business Hours)     Play message 0001.
                                Go to COLLECT ID NUMBER.

 If EP=001 (Holiday)            Play message 0002 (uninterruptible). 
                                End Call.

 If EP=other                    Play 0.22.705, 0.22.684 & 0.22.312.  
                                End call.


COLLECT ID NUMBER               Play message 0003.
                                Collect 10 digit number
                                (# is an optional delimiter)

  If 10 digits rec'd            Save 10 digits as ID_NUMBER
                                Go to ID NUMBER LOOKUP

  If Timeout
    1st time                    Play message 0004.
                                Go to COLLECT ID NUMBER
    2nd time                    Play message 0006.
                                Write "TO" to ID_NUMBER field of CCRs.
                                End call.

  If Invalid (input contains * or #)
    1st time                    Play message 0005.
                                Go to COLLECT ID NUMBER.
    2nd time                    Play message 0006.
                                Write "INV" to ID_NUMBER field of CCRs.
                                End call.


ID NUMBER LOOKUP                Using variable length ID_NUMBER as key,
                                search CM020680_HIM_Medicaid_NDB
                                table from CM020680/AppID 0020680 using
                                procedure "CM020680_HIM_Medicaid_query".

   If found                     Write "Found" to LOOKUP field of CCRs
                                Go to GENERATE CALL COUNTER

   If not found                 Write "Not found" to LOOKUP field of CCRs
                                Extend call to 800-318-2596 (use type 1 call extn)
                                RLT the call.

   If NDB unavailable/fails     Using variable length ID_NUMBER as key,
                                search DB18 table from CM020680/AppID 0020680.

       If found                 Write "Found" to LOOKUP field of CCRs
                                Go to GENERATE CALL COUNTER

       If not found             Write "Not found" to LOOKUP field of CCRs
                                Extend call to 800-318-2596 (use type 1 call extn)
                                RLT the call.


GENERATE CALL COUNTER           Use NDB procedure to generate a 4-digit counter.
                                Begin counter at 0001 and increment by 1 each
                                time NDB is accessed
                                (counter resets or rolls over to 0001 when it 
                                 reaches 9999).

 If successful                  Set COUNTER = 4-digit counter,
                                Go to CREATE CALL ID.

 If procedure fails or
 is unavailable                 Set COUNTER = '0000',
                                Go to CREATE CALL ID.


CREATE CALL ID                  Create a 26-digit JP_Call_ID for Jump Parameter 
                                & a 31-character CCR_Call_ID for CCR_String:
                                Set JP_Call_ID = CallStartDateTime + 
                                 sys.MSP.ANI.number + COUNTER
                                 (Example: YYMMDDHHMMSS71959912340001)
                                Set CCR_Call_ID = '0100x' + JP_Call_ID
                                 (Example: 0100xYYMMDDHHMMSS71959912340001)
                                Go to SKILL MENU                                         ***CHG 06/14/16***


SKILL MENU                      Play message 0008.                                       ***ADD 06/14/16***                                
If caller presses
  1 - Tech Assistance           Set SKILLCODE = "FFM_Spare1".
      (Hidden)                  Set SKILLCODE_CCR = "FFM_Spare1".
                                Go to FINALIZE CCR STRING

  2 - SEP Help                  Set SKILLCODE = "FFM_ET2".
                                Set SKILLCODE_CCR = "FFM_ET2".
                                Go to FINALIZE CCR STRING

  3 - Other Questions           Set SKILLCODE = "FFM_ET1".
                                Set SKILLCODE_CCR = "FFM_ET1".
                                Go to FINALIZE CCR STRING

If Invalid
  1st/2nd Time                  Play message 0005
                                Go to SKILL MENU

  3rd Time                      Play message 0006
                                Write "INV" to SKILLCODE_CCR field of CCRs.
                                End call.

If No Response
  1st/2nd Time                  Play message 0009
                                Go to SKILL MENU

  3rd Time                      Play message 0006
                                Write "TO" to SKILLCODE_CCR field of CCRs.
                                End call.


FINALIZE CCR STRING             Set SKILLCODE = "FFM_ET1"                                ***DEL 06/14/16***
                                Append CCR_String with ID_NUMBER, LOOKUP, 
                                 SKILLCODE_CCR and CCR_Call_ID
                                 NOTE: CCR fields are separated by commas
                                Go to JUMP TO MESSAGING APP.


JUMP TO MESSAGING APP           Play message 0007.
                                Jump to appID 019911, EP_000.
                                - Pass 26-digit JP_Call_ID in jump parameter 0.
                                - Pass SKILLCODE in jump parameter 10.                   ***CHG 06/14/16***
                                - Pass CCR_String in jump parameter 11.
                                - Pass "FFM25824" in jump parameter 15.
